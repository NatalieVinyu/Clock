# Clock
A simple digital clock using HTML, CSS, and JavaScript that displays the current time and updates automatically every second.

## Technologies used
* HTML
* CSS
* Javascript

## What it does:
* The clock displays the current hours, minutes and seconds
* Has a dark mode / light mode switch that changes the clock's appearance
* Has a button that toggles between 12 Hour and 24 Hour Formats
* The clock updates automatically
